Updated 2021-07-10 19:49:55
Version 3


[h3]
What happened to the HLDSUpdateTool? Why can't I download or update servers from it?
[/h3]
All Steam game servers now use SteamCMD to download and update game server files. HLDSUpdateTool has been retired and will no longer function.
[h3]
How do I setup and use SteamCMD to install and update my game servers?
[/h3]
For more information on installing and using SteamCMD, please see the [url=https://developer.valvesoftware.com/wiki/SteamCMD]Valve Developer Wiki[/url] which includes the latest information on SteamCMD.
[h3]
Where can I find a list of supported games?
[/h3]
For a list of dedicated servers and games that SteamCMD supports on Steam, please see the [url=https://developer.valvesoftware.com/wiki/Dedicated_Servers_List]Dedicated Servers List[/url].
You can also post questions and search for help in [url=http://steamcommunity.com/discussions/#tools]Steam Discussions[/url].
